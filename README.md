# Simple react application for searching github repositories with using github API

## Using technologies:
* Webpack
* React, React-hooks
* Axios
* Redux, react-redux
* Bootstrap
* Scss
* Fontawesome
* github REST API

### Installation:
```
$ npm i
```

### Run with devServer:
```
$ npm start
```
### To build in dev mode:
```
$ npm run dev
```
### For building production bundle:
```
$ npm run build
```
